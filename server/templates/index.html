Tahmatassu-API

Method: GET

Route: /index

Method: GET

Route: /recipe/:name

Return recipe with given :name

Response JSON:
{
   "markdown":"",
   "name":"Lihapullat.md",
   "modified":"2014-03-20 23:05:10",
   "created":"2014-03-20 20:17:55"
}

Method: GET

Route: /recipes

List names of all recipes, these names can be used to read fetch the recipes

Response JSON:
["Lihapullat.md", "Repesti.md","Desu.md"]

Method: PUT,POST

Route: /recipe

Store a recipe to service. Takes a JSON-data in body.

Response JSON:

   "markdown":"",
   "name":"Lihapullat",
   "modified":"2014-03-20 23:05:10",
   "created":"2014-03-20 20:17:55"
